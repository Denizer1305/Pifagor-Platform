Вопрос 7: Что выведет следующий код?

Неправильно
x = 5
y = 2
print(x // y)
Ваш ответ
2.5
Правильный ответ
2
Объяснение:

Оператор // в Python выполняет целочисленное деление, которое отбрасывает дробную часть и возвращает целое число. В отличие от оператора /, который выполняет обычное деление и возвращает число с плавающей точкой.
