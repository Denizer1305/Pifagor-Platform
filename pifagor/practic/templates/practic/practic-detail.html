Задание 1: Работа с переменными

Научитесь создавать переменные и работать с основными типами данных в Python

15-25 минут
Начальный уровень
Python

Описание задания

В этом задании вы научитесь создавать переменные и работать с основными типами данных в Python. Вам нужно будет создать несколько переменных, выполнить с ними операции и вывести результат.

Требования к решению:
  • Создайте переменную name и присвойте ей ваше имя (строка)
  • Создайте переменную age и присвойте ей ваш возраст (целое число)
  • Создайте переменную height и присвойте ей ваш рост в метрах (дробное число)
  • Создайте переменную is_student и присвойте ей значение True (логическое значение)
  • Выведите все переменные на экран в формате: "Меня зовут [name], мне [age] лет, мой рост [height] м, студент: [is_student]"
Пример вывода:
Меня зовут Анна, мне 20 лет, мой рост 1.65 м, студент: True
Редактор кода
Python
