Детали выполнения

В этом задании вам предстояло решить квадратные уравнения различными методами. Квадратное уравнение — это уравнение вида ax² + bx + c = 0, где a, b, c — коэффициенты, причём a ≠ 0.

Требования к решению:
  • Решите 10 квадратных уравнений, приведенных ниже
  • Для каждого уравнения укажите использованный метод решения
  • Предоставьте полное решение с пояснениями каждого шага
  • Проверьте корни, подставив их в исходное уравнение
  • Оформите решение аккуратно и разборчиво
