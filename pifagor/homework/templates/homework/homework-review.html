1. Уравнение: x² - 5x + 6 = 0 Решение: D = b² - 4ac = (-5)² - 4*1*6 = 25 - 24 = 1 x₁ = (5 + 1) / 2 = 3 x₂ = (5 - 1) / 2 = 2 Ответ: x₁ = 3, x₂ = 2 2. Уравнение: 2x² + 3x - 2 = 0 Решение: D = 3² - 4*2*(-2) = 9 + 16 = 25 x₁ = (-3 + 5) / 4 = 0.5 x₂ = (-3 - 5) / 4 = -2 Ответ: x₁ = 0.5, x₂ = -2 3. Уравнение: x² - 4x + 4 = 0 Решение: D = (-4)² - 4*1*4 = 16 - 16 = 0 x = 4 / 2 = 2 Ответ: x = 2 4. Уравнение: x² + 2x + 5 = 0 Решение: D = 2² - 4*1*5 = 4 - 20 = -16 Ответ: Действительных корней нет 5. Уравнение: 3x² - 7x + 2 = 0 Решение: D = (-7)² - 4*3*2 = 49 - 24 = 25 x₁ = (7 + 5) / 6 = 2 x₂ = (7 - 5) / 6 = 1/3 Ответ: x₁ = 2, x₂ = 1/3 ... и так далее до 8 уравнения

Задание: Решение квадратных уравнений Решите 10 квадратных уравнений различными методами. Для каждого уравнения предоставьте полное решение с пояснениями каждого шага. Проверьте корни, подставив их в исходное уравнение. Уравнения: 1. x² - 5x + 6 = 0 2. 2x² + 3x - 2 = 0 3. x² - 4x + 4 = 0 4. x² + 2x + 5 = 0 5. 3x² - 7x + 2 = 0 6. x² - x - 6 = 0 7. 4x² - 12x + 9 = 0 8. x² + 6x + 9 = 0 9. 2x² - 5x - 3 = 0 10. x² + 4x + 13 = 0 Требования: - Решите все 10 уравнений - Для каждого уравнения укажите использованный метод решения - Предоставьте полное решение с пояснениями каждого шага - Проверьте корни, подставив их в исходное уравнение - Оформите решение аккуратно и разборчиво Критерии оценки: - Правильность решения уравнений (5 баллов) - Полнота объяснений (3 балла) - Аккуратность оформления (2 балла)
