Решение квадратных уравнений

Описание задания

В этом задании вам предстоит решить квадратные уравнения различными методами. Квадратное уравнение — это уравнение вида ax² + bx + c = 0, где a, b, c — коэффициенты, причём a ≠ 0.

Требования к решению:
  • Решите 10 квадратных уравнений, приведенных ниже
  • Для каждого уравнения укажите использованный метод решения
  • Предоставьте полное решение с пояснениями каждого шага
  • Проверьте корни, подставив их в исходное уравнение
  • Оформите решение аккуратно и разборчиво
Пример уравнения для решения:
Уравнение: 2x² - 5x + 3 = 0 Решение: 1. Найдем дискриминант: D = b² - 4ac = (-5)² - 4*2*3 = 25 - 24 = 1 2. Так как D > 0, уравнение имеет два корня: x₁ = (5 + √1) / (2*2) = (5 + 1) / 4 = 6 / 4 = 1.5 x₂ = (5 - √1) / (2*2) = (5 - 1) / 4 = 4 / 4 = 1 3. Проверка: Для x=1.5: 2*(1.5)² - 5*1.5 + 3 = 2*2.25 - 7.5 + 3 = 4.5 - 7.5 + 3 = 0 Для x=1: 2*(1)² - 5*1 + 3 = 2 - 5 + 3 = 0 Ответ: x₁ = 1.5, x₂ = 1
