Лекция: Введение в условные операторы

Что такое условные операторы?

Условные операторы позволяют программе принимать решения на основе определенных условий. В Python для этого используются операторы if, elif и else.

Базовый синтаксис оператора if

Простейшая форма условного оператора выглядит так:

if условие:
    # код, который выполнится, если условие истинно
    выражение

Пример использования if

Рассмотрим простой пример:

age = 18
if age >= 18:
    print("Вы совершеннолетний")

Операторы if-else

Когда нужно предусмотреть альтернативный вариант, используется конструкция if-else:

age = 16
if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

Множественные условия с elif

Когда нужно проверить несколько условий, используется оператор elif:

score = 85
if score >= 90:
    print("Отлично!")
elif score >= 70:
    print("Хорошо")
elif score >= 50:
    print("Удовлетворительно")
else:
    print("Неудовлетворительно")

Логические операторы

Для создания сложных условий используются логические операторы:

  • and - логическое И (оба условия истинны)
  • or - логическое ИЛИ (хотя бы одно условие истинно)
  • not - логическое НЕ (инвертирует условие)
age = 25
has_license = True

if age >= 18 and has_license:
    print("Вы можете водить машину")
else:
    print("Вы не можете водить машину")
